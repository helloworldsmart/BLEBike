# BLEBike

`BLEBike`是靠腳踏車上藍牙裝置紀錄騎乘數據一套App，使用方便、容易上手、並且直觀讓你紀錄運動狀態

### 特色
-----------
	1. 騎乘數據Dashboard - 直觀清楚的顯示BLEBike功率心律等資訊
	
	2. 設定地圖訓練目標 - 直觀滑動轉換成地圖頁面
	
	3. Realm - 永久性儲存資料
	
	4. Charts - 統計數據圖                                                                                                                                                                                                                                                               5. Core Bluetooth - 用Nordic板子BLE串接
